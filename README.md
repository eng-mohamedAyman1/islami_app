# Islami App

A clean and simple Islamic mobile app built with **Flutter**, offering daily Islamic content — all available **offline**.

---

##  Features

###  Onboarding
- Shown only on first launch
- State saved with SharedPreferences

###  Quran
- Search Surah by name
- View last 5 recently opened Surahs
- Browse all 114 Surahs with full details

###  Hadith
- Swipe through 50 Hadith using a carousel
- Tap any Hadith to read full details

> Tabs 4 & 5 are not yet implemented.
---

## 🛠 Tech Stack

**Flutter / Dart** — local data, `setState` for state management, `Navigator` for routing.

**Packages:**
`flutter_native_splash` · `introduction_screen` · `shared_preferences` · `flutter_svg` · `lottie` · `flutter_screenutil` · `carousel_slider`
